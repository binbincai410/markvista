# MarkVista

本地 Markdown 阅读工具，支持 Mermaid 图渲染与缩放平移，可导出整篇为 PDF 或 PNG 长图。

## 1. 安装依赖

```bash
npm install
```

## 2. 启动

```bash
npm run dev
```

## 3. 访问地址

终端会输出本地地址，例如：

```
http://localhost:5173
```

在浏览器中打开该地址即可使用。

## 功能说明

- **选择文件**：选择本地 `.md` 文件并渲染
- **刷新**：重新读取当前文件内容并重新渲染
- **导出 PDF**：整篇导出为 PDF（浏览器打印）
- **导出 PNG**：整篇导出为 PNG 长图（2x 分辨率，文件名含时间戳）

Mermaid 代码块支持滚轮缩放与拖拽平移。
